## QtMultiDir

- 作者：李长圣
- 邮箱：sheng0619@163.com
- 功能：演示如何使用 `qtcreator` 创建 app+lib 多目录程序
- 使用方法：使用 `qtcreator` 打开 `QtMultiDir.pro` 编译运行
- 支持 `window10` 系统

### 操作方法
  
* 打开 `QtMultiDir.pro`，运行。输出
    ```
    Hello world!
    ```
